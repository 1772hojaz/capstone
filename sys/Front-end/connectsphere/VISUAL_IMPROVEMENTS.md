## ğŸ–±ï¸ Interactive States

### Buttons:
1. **Default**: Gradient, shadow

2. **Hoverable**: Lift on hover, shadow increase

### Inputs:
1. **Default**: Gray border

3. **Error**: Red border + ring + error icon
4. **Success**: Green border + ring
5. **Disabled**: 50% opacity
